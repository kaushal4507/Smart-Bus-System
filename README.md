# Smart Bus System

A web-based Bus Management System developed using Java, JSP, Servlets, JDBC, Oracle SQL, HTML, CSS, and JavaScript.

## Features

- User Login and Registration
- Bus Route Management
- Ticket Booking
- Passenger Management
- Schedule Management

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- Oracle SQL
- HTML
- CSS
- JavaScript
- Apache Tomcat
